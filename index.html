PDF Chapter Splitter

Split large PDFs into smaller chapter-based files - perfect for working with AI tools that have token limits.

  • Automatic chapter detection
  • Adjustable splitting depth
  • Download results as ZIP

How to Use:

  1. Upload your PDF using drag & drop or file dialog
  2. Choose how deep you want to split (main chapters only, include subchapters, etc.)
  3. Click "Split PDF"
  4. Download your ZIP file containing individual chapter PDFs
  5. Use these smaller PDFs with AI tools that have token limits

Note: Your PDF must have a table of contents/outline for splitting to work.

PDF Page Stripper

Clean up incrementally built presentation slides by removing intermediate steps.

  • Multiple comparison modes
  • Batch processing
  • Keeps only final slides

How to Use:

  1. Upload your presentation PDF(s)
  2. Choose comparison mode:
    • Word Comparison: Strips more pages by comparing at word level
    • Full Text Comparison: More conservative, compares entire text
  3. Process files
  4. Download clean PDFs with only final slides

Best for: Presentations with step-by-step slide builds or animations
