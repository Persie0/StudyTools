AI-Powered PDF to Anki Converter

How it works:

  1. Select an AI provider and enter one or more API keys (one per line). (You can save these to your browser).
  2. Upload a single PDF file.
  3. The app sends your PDF (or its text, per your settings) to the selected AI model to get a JSON list of grouped slides.
  4. Choose a mode:
    • Default Mode: For each *group*, it sends the text to the selected AI again to get a detailed explanation for the "BackContent" field.
    • Annotation Mode: For *each slide* in a group, it asks the selected AI for a brief annotation *if needed*. This is added below the slide image in the "Back" field.
  5. It then builds an Anki deck with the slide images and AI explanations.

⚠️ Warning: This tool makes many API calls and may be slow. **Slide-by-slide mode makes significantly MORE calls.**

✅ New: The tool automatically retries on rate limits or 503 (model overloaded) errors. If all your keys are busy, it will show a **countdown timer** and wait for the soonest one to be available.
